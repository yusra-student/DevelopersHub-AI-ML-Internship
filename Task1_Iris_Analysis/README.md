# Task 1: Exploring and Visualizing the Iris Dataset

## Overview
This repository contains the first task of my AI/ML Engineering Internship at DevelopersHub Corporation. The project focuses on Exploratory Data Analysis (EDA) of the classic Iris dataset using Python.

## Project Structure
```text
Task1_Iris_Analysis/
├── notebook/
│   └── iris_analysis.ipynb     # Main analysis notebook
├── plots/                      # Generated visualization files
│   ├── scatter_plot.png
│   ├── histograms.png
│   ├── boxplots.png
│   └── pairplot.png
├── requirements.txt            # Project dependencies
└── README.md                   # Project documentation
```

## Problem Statement
The goal is to perform a comprehensive exploration of the Iris dataset to understand the characteristics of three different species of Iris flowers (Setosa, Versicolor, and Virginica) based on their sepal and petal measurements.

## Key Features
- **Data Inspection**: Comprehensive look at data types, statistics, and distributions.
- **Data Visualization**: High-quality plots including Scatter, Histograms, Box Plots, and Pair Plots.
- **Analytical Insights**: Detailed markdown cells explaining the findings from each visualization.

## How to Run
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Open the Jupyter Notebook:
   ```bash
   jupyter notebook notebook/iris_analysis.ipynb
   ```
